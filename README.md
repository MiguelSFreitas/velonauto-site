# Velon Auto 🚗📰

O **Velon Auto** é um portal de notícias digital dedicado ao universo automotivo. O projeto nasceu com o propósito de entregar aos entusiastas de carros, compradores e profissionais do setor um conteúdo jornalístico dinâmico, visualmente atraente e de alta qualidade. Cobrimos desde os lançamentos mais aguardados e tendências tecnológicas (como veículos elétricos e híbridos) até segredos de mercado e avaliações completas.

---

## 🎯 Objetivo do Projeto

Este repositório armazena o ecossistema de código do portal. O foco principal do desenvolvimento é criar uma plataforma web moderna, rápida, totalmente responsiva e otimizada para motores de busca (SEO), garantindo que os leitores tenham a melhor experiência de leitura em qualquer dispositivo (celular, tablet ou computador).

---

## 🛠️ Tecnologias e Conceitos Utilizados

* **HTML5 Semântico:** Uso rigoroso de tags estruturais (`<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<footer>`) para garantir acessibilidade e uma indexação limpa no Google.
* **CSS3 Moderno:** Arquitetura de estilos padronizada, focada em tipografia legível, identidade visual marcante e layout fluido.
* **Design Orientado ao Conteúdo:** Telas limpas, espaços bem distribuídos e foco total na escaneabilidade da notícia.
